ACME Solutions is a dummy company created to showcase front-end development skills. This site is built on October CMS with Twig templates, styled with Tailwind CSS, and enhanced with Alpine.js micro-interactions.

Why this exists

The goal is to demonstrate a complete development workflow: from local design iteration with hot-reload, through version control, to automated deployment on Google Cloud Platform. Every page showcases a different aspect of front-end craft.

The stack

October CMS handles content and templating. Tailwind CSS provides a token-based design system. Alpine.js adds lightweight interactivity. Laravel Mix compiles assets. Docker packages the app. GitHub Actions deploys to GCP Cloud Run in Mumbai.
